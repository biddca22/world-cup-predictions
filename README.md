# world-cup-predictions
Predictive modelling of 2002-2022 FIFA World Cup matches with pre-match and in-match data. Includes outcome prediction, win-probability estimation, and statistical analysis.

## Data Sources

* Zippenfenig, P. (2023). Open-Meteo.com Weather API. [doi.org/10.5281/ZENODO.7970649](https://doi.org/10.5281/ZENODO.7970649)
* Hersbach, H., et al. (2023). ERA5 hourly data on single levels from 1940 to present. ECMWF. [doi.org/10.24381/cds.adbb2d47](https://doi.org/10.24381/cds.adbb2d47)
